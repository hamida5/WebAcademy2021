What is Inline CSS?

Inline CSS allows you to apply style rules to HTML elements.Inline CSS usage is limited but it is beneficial for creating unique attributes. It can be used to apply a unique style of a single HTML element. Inline CSS is useful for quick and permanent changes, however,. they are less flexible than external and internal CSS because if you are making a chnage to the style, you must edit each one seperatley.

What is Internal CSS?

Internal CSS is used to add a unique style for a single document. It can only be used in one single HTML page. Several pages can be styled by repeating the same block of internal styles in them. Internal CSS is defined inside the style element. A single change to the CSS rule will apply to all tagged page elements.

What is External CSS?

External CSS is a seprate file that can be used by craeting a link in the head section of a webpage. Multiple webpages can use the same link. External CSS can be used in any text editor, but it must be saved with a .css extension. It should not contain any HTML tags. External CSS is benefical because if you change something in the style sheet, all the link pages will also change. This saves a lot of time and work. It also creates a consistant look across multiple webpages. This makes the webpage look nice and neat.
