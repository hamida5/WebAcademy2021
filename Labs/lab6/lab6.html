Favicon, Banner, and Branding


What exactly is a favicon?

A favicon is a small square image that is displayed in a browser's address bar or next to the site name in a bookmark list. Favicons are usually logos that serve as a branding for your website. They can be found on almost all websites. They help locate a page easier.

How are favicons created?

To create a favicon, we first have to create an icon or image. We can do this by using Fireworks, Photoshop, Corel Paint, Code, Photopea, or any other website. We save our image as a png or jpg. After we create our image, we can can convert the image into favicon.ico. Next we upload the image to out website. Lastly, we can add the basic HTML code, (link rel="shortcut icon" href="" type="image/x-icon").We use the code (link rel="shortcut icon" href="" type="image/x-icon"). Now you can find your favicon on your browser's address bar!

How did I create the favicon for this site?

I created my favicon by using Code.org. After I created a basic draft of my favicon on graph paper, I started to make a black and white version. I used binary to create the black and white pixels. 1=white and 0=black. My image was 32 x 32 pixels. After creating my black and white image, I moved on to creating a colorful one. It was the same process except I had to use the color bits code. I did six bits per pixel and used 3 colors. After I was satisfied with my work I saved it as a png file. I unploaded it to replit and used (link rel="shortcut icon" href="" type="image/x-icon") to insert it to my webpage.
